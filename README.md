PSG - Tecnologia Aplicada
==========================

O objetivo dessa aplicação é contabilizar a quantidade de falhas em uma plantação de cana-de-açúcar a partir de imagens aéreas obtidas através de aeronaves não-tripuladas, como VANTs.
